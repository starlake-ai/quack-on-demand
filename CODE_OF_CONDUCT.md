# Code of Conduct

This project has adopted the [Contributor Covenant, version 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/),
as our code of conduct. The full text is reproduced in the link above
and is the authoritative version.

## Scope

It applies in all project spaces (the GitHub repository, issue tracker,
pull requests, discussions, and any other community space that uses
the project name) and when an individual is officially representing
the project in public spaces.

## Reporting

Instances of behaviour that conflict with the Covenant can be reported
privately to the project maintainers at **hayssam.saleh@starlake.ai**.
All reports are reviewed and investigated; reporter identity is kept
confidential.

## Enforcement

Maintainers will follow the [Community Impact Guidelines](https://www.contributor-covenant.org/version/2/1/code_of_conduct/#enforcement-guidelines)
from the Covenant when responding to reported incidents. Consequences
range from a private warning to a permanent ban from project spaces.
